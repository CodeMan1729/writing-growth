# 例题1
>[!note] 李扬-26年强化讲义-例题1.1.3
>$a_1=\sqrt{2},a_{n+1}=\sqrt{2+a_n}$，求极限$$\lim_{n\to \infty}\frac{2^n}{a_1a_2...a_n}$$

1. 使用mathematical画出图像以后发现它的极限值是介于1和2之间，并不能帮我猜测极限，但可以看到数列是单调递增有极限的
2. 通过不断的尝试，发现如果把极限先倒过来求，令$b_{n+1}=\frac{a_{n+1}}{2}$，那么$b_{n+1}=\sqrt{\frac{1+b_n}{2}}$,根据[[三角代换#^aa4a92]]，考虑令$b_n=cos(\frac{\theta}{2^n})$，这样$b_{n+1}=cos(\frac{\theta}{2^{n+1}})$
3. 但要考虑存不存在这样的映射，满足$f$是双射：$$f:(b_n,n)\to(\theta,n)$$，因为$\frac{\sqrt{2}}{2}\leq b_n<1$，所以一定有$(\theta,n)$满足，反之亦然。故，存在这样的映射
4. 再根据极限
