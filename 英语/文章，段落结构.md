# 文章结构
### 2009-43,44
按照七选五的做题办法，我们发现43和44都可以填A跟B
所以我们画出涉及段落的文章结构

```
boas相关
🔽
(43)
🔽
boas相关
🔽
(44)
🔽
boas成为研究主流
diffusion

A. diffusion相关
B. boas相关
```

如果填写A->B，文章是总分结构
B->A，44前面都是boas，后面突然插入diffusion，然后又开始说boas，不合逻辑

因为最上面的一段出现了两个对象的逻辑连接词，我们可以看到都有known as（A中），更加坚定是总分结构

# 过渡句

### 2009-45
45题的上面是说大部分的文化成就都是由少数的天才制作出来的，然后传播。是一个论点，根据[[论点-论据原则]]，所以45应该是一个支撑他的论据。但是我们还要看七选五后面一句话，后面一段的第一句话是说有有一个人提出了一种新的理论那之所以提出一种新的理论正是因为前一种理论他不是那么完善所以这个例子中也应该包含这个diffusion理论不对的地方这样G才可以作为一个过渡句


综合以上两点，选择G
